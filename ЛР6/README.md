# Имитация отжига для решения квадратичного уравнения
Задача: использовать имитацию отжига для нахождения минимума функции f(x) = (x-7)² + 5.
Требования:
- Начальное решение: случайное число из [0, 20]
- Соседнее решение: добавить случайный сдвиг к текущему
- Вывести найденный минимум и его координату
Входные данные: Диапазон поиска: [0, 20]
